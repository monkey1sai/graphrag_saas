# TWIBM RAG 條目：0021

- 目標對象：求職應徵 (求職者)
- 第一層關鍵詞（主旨）：績效管理與職涯發展
- 第二層關鍵詞（細項）：多層次考核機制

## 詳細說明（條列）
- 實施月度、季度及年度考核
- 包含關鍵績效指標(KPI)檢視
- 分數低於標準者需再訓練或淘汰

## 出處原文（RAG 依據）
透過每月例行區處會議評核...每三個月實施品質績效評核檢視...年度實施管理績效評核作業...未達85分以上者不予以錄用

## 預測問題（QA Generation）
1. 公司的績效考核頻率為何？
2. 關鍵績效指標(KPI)如果不合格會如何處理？
3. 考核結果會影響錄用資格嗎？
