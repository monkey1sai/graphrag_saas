# TWIBM RAG 條目：0025

- 目標對象：求職應徵 (求職者)
- 第一層關鍵詞（主旨）：作業標準與職業規範
- 第二層關鍵詞（細項）：緊急應變標準作業程序(SOP)

## 詳細說明（條列）
- 針對火災、電梯受困、水災、怪客入侵及醫療急救等狀況
- 建立詳細的處置流程與通報機制

## 出處原文（RAG 依據）
緊急應變處置...電梯乘廂內緊急呼叫...如遇火災...如遇水災...對陌生或神情可疑之訪客...住戶使用公設受傷

## 預測問題（QA Generation）
1. 遇到火災或地震時的標準作業程序是什麼？
2. 若住戶受困電梯，員工應如何處置？
3. 針對可疑訪客或怪客入侵有哪些應變機制？
