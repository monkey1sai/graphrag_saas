# TWIBM RAG 條目：0036

- 目標對象：求職應徵 (求職者)
- 第一層關鍵詞（主旨）：環境維護職務
- 第二層關鍵詞（細項）：清潔維護人員

## 詳細說明（條列）
- 職責：執行週期性清潔計畫、資源回收分類、特殊材質(石材/木質/鍍鈦)保養、景觀水池清潔。
- 訓練：各類建材清潔方式(17項)、工具使用(24項)、禮儀訓練(早安/您好/辛苦了)、除膠除油技術。

## 出處原文（RAG 依據）
清潔維護人員...排定每年及每月、每日清潔維護工作計劃表...環保室資源回收廢棄物...建材清潔方式及注意事項...鍍鈦專用清潔亮光劑
