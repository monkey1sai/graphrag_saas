# TWIBM RAG 條目：0030

- 目標對象：求職應徵 (求職者)
- 第一層關鍵詞（主旨）：安管與保全職務
- 第二層關鍵詞（細項）：車道入口/出口哨

## 詳細說明（條列）
- 職責：車道交管指揮、施工車輛登記換證、司機休息室管理、車輛離場檢查、違規停車勸導。
- 訓練：交管手勢與動作、法律常識、施工車輛管理流程、消防設施設備操作。

## 出處原文（RAG 依據）
車道入口/出口哨...負責車道出入口交管...負責施工車輛之登記、換證...負責貴賓/住戶車輛離開時之交管與指揮

1. 車道哨保全需要負責哪些交通管制工作？
2. 對於施工車輛進出有哪些管理流程？
3. 車道哨是否需要管理司機休息室？
