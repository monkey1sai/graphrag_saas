# TWIBM RAG 條目：0022

- 目標對象：求職應徵 (求職者)
- 第一層關鍵詞（主旨）：績效管理與職涯發展
- 第二層關鍵詞（細項）：明確的晉升階梯

## 詳細說明（條列）
- 提供透明的升遷管道
- 基層保全可透過累積經驗、通過考核及取得證照
- 晉升為物業經理或處級主管

## 出處原文（RAG 依據）
保全同仁晉升物業經理...須領有事務管理證照外...通過管理績效評核...由區處主管評核會議考核通過後，使得升任主管
