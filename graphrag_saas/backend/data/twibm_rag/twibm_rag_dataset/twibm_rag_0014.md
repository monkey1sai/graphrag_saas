# TWIBM RAG 條目：0014

- 目標對象：求職應徵 (求職者)
- 第一層關鍵詞（主旨）：職務專業資格與職責
- 第二層關鍵詞（細項）：社區經理/高階主管

## 詳細說明（條列）
- 需具備事務管理、設備安全、防火管理等國家證照
- 職責涵蓋年度計畫、財務控管、公共關係建立及緊急狀況指揮

## 出處原文（RAG 依據）
任用專業證照資格...如事務管理、設備、防火管理證照...社區經理工作職掌...每日批閱社區九大簿冊...年度管理工作計劃制定執行
