# TWIBM RAG 條目：0017

- 目標對象：求職應徵 (求職者)
- 第一層關鍵詞（主旨）：職務專業資格與職責
- 第二層關鍵詞（細項）：安管與保全人員

## 詳細說明（條列）
- 需領有保全護照
- 分為迎賓哨(禮賓接待)、車道哨(車輛進出管制)及中控哨(監控系統操作)

## 出處原文（RAG 依據）
領有保全業法規定之保全護照...迎賓哨...負責大廳前廊警戒...車道入口/出口哨...負責車道出入口交管...中控保全人員...CCTV監視錄影系統螢幕監視
