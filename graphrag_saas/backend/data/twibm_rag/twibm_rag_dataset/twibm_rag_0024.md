# TWIBM RAG 條目：0024

- 目標對象：求職應徵 (求職者)
- 第一層關鍵詞（主旨）：作業標準與職業規範
- 第二層關鍵詞（細項）：標準化服務禮儀

## 詳細說明（條列）
- 制定詳細的儀態規範
- 包括站姿、行走、引導手勢及應對進退禮節
- 要求做到「溝通零距離」

## 出處原文（RAG 依據）
站立要領...上身正直、挺胸收腹...行走要領...昂首、挺胸...引導...走在賓客的左前方...溝通零距離，服務無止境
