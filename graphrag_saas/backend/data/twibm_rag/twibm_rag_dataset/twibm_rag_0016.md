# TWIBM RAG 條目：0016

- 目標對象：求職應徵 (求職者)
- 第一層關鍵詞（主旨）：職務專業資格與職責
- 第二層關鍵詞（細項）：專業秘書群 (行政/吧台/俱樂部)

## 詳細說明（條列）
- 依職能分工：吧台秘書需習得拉花與菜單設計
- 俱樂部秘書負責公設預約
- 行政秘書處理財務與文書
- 需熟電腦與禮賓技巧
